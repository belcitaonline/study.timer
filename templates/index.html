    Como enlazar tu cuenta?

    Como este sitio es estatico, Spotify no permite iniciar sesion de forma directa de manera segura. Usamos Last.fm como puente gratuito:

    1. Crea un usuario en last.fm.
    2. Ve a Configuracion y en Aplicaciones dale a Conectar en Spotify. Esto leera lo que escuches en cualquier dispositivo.
    3. Entra a last.fm/api/account/create para obtener un codigo llamado API Key.
    4. Abre este archivo HTML con un editor y pon tu usuario y API Key en las variables del script de abajo.
